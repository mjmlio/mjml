
## MJ-BODY

``` html

<mj-body>

  <!-- You email goes here -->

</mj-body>

```

This is the starting point of your email. It is a unique and mandatory component.
It corresponds to the HTML `<body>` tag.

Everything outside the `<mj-body>` will not be parsed by the engine.
