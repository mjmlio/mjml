## mjml-chart

Displays a image chart in your email. Note that the chart can be animated (gif) if the `chan` attribute is specified.

```xml
<mjml>
  <mj-body>
    <mj-container>
      <mj-section>
        <mj-column>
          <mj-chart chs="300x200" chd="t:10,20,30|15,25,35" cht="bvs" chxt="x,y" chxl="0:|A|B|C" />
        </mj-column>
      </mj-section>
    </mj-container>
  </mj-body>
</mjml>
```
