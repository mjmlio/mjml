## mjml-location

  <img src="https://cloud.githubusercontent.com/assets/2217014/13528219/d574b84c-e214-11e5-8504-f9cd250b65fa.png" />

Display a Google Maps location link

```xml
<mjml>
  <mj-body>
    <mj-container>
      <mj-section>
        <mj-column>
          <mj-location address="37 bis, rue du Sentier 75002 Paris" />
        </mj-column>
      </mj-section>
    </mj-container>
  </mj-body>
</mjml>
```
