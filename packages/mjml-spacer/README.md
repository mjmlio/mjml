## mjml-spacer

Displays a blank space.

```xml
<mjml>
  <mj-body>
    <mj-container>
      <mj-section>
        <mj-column>
          <mj-text>A first line of text</mj-text>
          <mj-spacer height="50px" />
          <mj-text>A second line of text</mj-text>
        </mj-column>
      </mj-section>
    </mj-container>
  </mj-body>
</mjml>
```
