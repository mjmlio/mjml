## MJ-TEXT

 ``` html

<mjml>
  <mj-body>
    <mj-container>
      <mj-section>
        <mj-column>

          <mj-text>
            <h1>Hey Title!</h1>
          </mj-text>

        </mj-column>
      </mj-section>

      <mj-section>
        <mj-column>

          <mj-text>
            Hey There!
          </mj-text>

        </mj-column>
      </mj-section>
    </mj-container>
  </mj-body>
</mjml>
 ```

This tag allows you to display text in your email.

<aside class="notice">
`MjText` can contain any HTML tag with any attributes. Don't forget to encode special characters to avoid unexpected behaviour from MJML's parser
</aside>
